# 📊 E-commerce Sales Dashboard (Power BI)

## 🔍 Overview
This project focuses on analyzing e-commerce sales data to uncover key business insights related to revenue, customer behavior, and order performance.

The dashboard is designed to help stakeholders make data-driven decisions through interactive visualizations.

---

## 🎯 Objectives
- Analyze revenue trends over time
- Understand customer distribution across states
- Identify top-performing product categories
- Evaluate order status and delivery performance

---

## 📌 Key Insights
- 📈 Revenue shows a steady growth trend over time
- 📦 Majority of orders are successfully delivered (~97%)
- 🛍 Few product categories contribute most of the revenue
- 🌍 Certain states generate higher sales compared to others

---

## 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization
